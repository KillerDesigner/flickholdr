Flickholdr

This dummy image generator is the best way to beautify your in development projects. It allows you to pull photos from flickr, based on the tags you choose.

They are chosen from the "interesting" images on Flickr which ensures you get some of the most beautiful imagery available on the web.

Flickholdr only gets photos with a creative commons license and adds proper attibution on each image, making sure you can use them in any context

Using flickholdr

It's very easy: simply use a special src attribute in your images, which you can get on the image chooser in flickholdr.com For example this html code <img src="http://flickholdr.com/640/360/sunrise,ocean" /> will generate the following image:
